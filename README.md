## Deploying-AI-via-AzureDevOps
This project focuses on deploying a Python-based AI solution within an eCommerce platform to enhance user experience through intelligent search and conversational assistance.
The system includes two key AI components:
- AI Smart Search
- AI Chatbot Assistant
From a DevOps perspective, the project implements an end-to-end CI/CD pipeline using Azure DevOps, Docker, and AWS EC2 (Linux). The application is containerized and built via automated pipelines, with Docker images pushed to Docker Hub.

Earlier, deployments were handled manually, which was time-consuming and error-prone. This project introduces automation where code changes are now automatically built and deployed to the development environment through the CI/CD pipeline.

Overall, this project demonstrates hands-on experience in combining AI-driven features with modern DevOps practices, including CI/CD automation, containerization, and cloud-based deployments.

## Tech Stack
Languages: Python
Frameworks: Django
AI Components: NLP-based Smart Search, Chatbot Assistant
DevOps Tools: Azure DevOps, Docker
Cloud Platform: AWS (EC2)
Database: PostgreSQL
Version Control: Git, GitHub
OS/Environment: Linux (Ubuntu)
